Use CSS in HTML
---------------------
1. Inline CSS    (style attribute)
    <div style='color':red> This is CSS div </div>
    limitation: Restriceted to only one element tag
2. Internal CSS  (Style tag/attribute)  (Restricted to one HTML page)
3. external CSS - Using link 
    Write in separate CSS file and import it in HTML file using link tag
        <link rel = stylesheet" href="demo_1.css">
